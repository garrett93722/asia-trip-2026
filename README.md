# 四人亚洲之旅网站（V6）

这是一个可直接部署到 GitHub Pages 的静态旅行网站。

## 文件说明

- `index.html`：网页主体
- `style.css`：页面样式和手机适配
- `script.js`：交互功能
- `trip-data.js`：旅行数据，后续主要修改这个文件
- `README.md`：使用说明

## 本地查看

双击 `index.html` 即可在浏览器中打开。

由于页面使用了 Google Fonts，首次打开时需要联网加载字体；字体加载失败也不影响使用。

## 修改内容

绝大多数旅行信息都集中在 `trip-data.js`：

- `cities`：四座城市信息
- `itinerary`：每日行程
- `flights`：航班
- `hotels`：酒店
- `checklist`：行前准备

请不要在公开仓库里填写：

- 护照号、身份证号
- 完整订单号
- 机票或酒店二维码
- 银行卡信息
- 保险单敏感号码

## 发布到 GitHub Pages

1. 登录 GitHub，新建一个公开仓库，例如 `asia-trip-2026`
2. 将本文件夹中的全部文件上传到仓库根目录
3. 打开仓库 `Settings`
4. 在左侧选择 `Pages`
5. `Build and deployment` 中选择 `Deploy from a branch`
6. Branch 选择 `main`，文件夹选择 `/ (root)`
7. 保存后等待页面生成
8. 访问 GitHub 给出的链接，例如：
   `https://你的用户名.github.io/asia-trip-2026/`

## 当前交互功能

- 手机和电脑自适应
- 城市筛选
- 每日行程展开/收起
- 深色模式
- 行前清单勾选
- 勾选状态保存在当前设备
- 今日行程跳转
- 打印/PDF 样式

注意：行前清单状态使用浏览器本地存储，不同设备之间不会同步。
